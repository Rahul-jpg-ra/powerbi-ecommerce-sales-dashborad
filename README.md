# E-Commerce Sales Dashboard

## Overview
This project is an interactive Power BI dashboard built to analyze e-commerce sales performance and generate business insights.

## Dashboard Features
- Total Revenue Analysis
- Monthly Revenue Trend
- Category-wise Revenue Analysis
- Top Revenue Generating Products
- Top Performing Cities
- Interactive Filters

## Tools Used
- Power BI
- DAX
- Data Visualization

## Key Metrics
- Revenue: ₹6.16M
- Orders: 1K
- Quantity Sold: 1K
